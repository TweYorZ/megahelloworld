# megahelloworld
MEGA HELLO WORLD IN RUS &amp; ENG!

# Install:
npm i megahelloworld

# Using:
```javascript
let helloworld = require("megahelloworld")

let { text, rusText } = await helloworld() 

console.log(text + ` ` + rusText) // consoling english and russian hello world 
```
